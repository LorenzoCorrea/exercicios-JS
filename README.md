# 🚀 Curso JS do Zero — Trilha de Exercícios

Trilha prática de JavaScript: cada módulo tem uma explicação curta e exercícios.
**A regra é: ler pouco, codar muito.**

## Ordem de estudo

| # | Módulo | O que você aprende |
|---|--------|--------------------|
| 1 | `01-variaveis-e-tipos.js` | let, const, string, number, boolean |
| 2 | `02-operadores-e-condicionais.js` | matemática, comparação, if/else |
| 3 | `03-loops.js` | for, while, repetição |
| 4 | `04-arrays.js` | listas, push, índices, percorrer |
| 5 | `05-objetos.js` | chave/valor, acessar e alterar propriedades |
| 6 | `06-funcoes.js` | function, parâmetros, return, arrow |
| 7 | `07-dom/` | JS no navegador: querySelector, eventos |

## Como rodar

**Opção A — Node (mais rápido):** abra o terminal na pasta e rode:
```
node 01-variaveis-e-tipos.js
```

**Opção B — Navegador:** abra o `index.html` desta pasta com o Live Server
e veja o resultado no Console (F12 → aba Console). Para trocar de módulo,
mude o `src` do `<script>` no `index.html`.

O módulo 7 (DOM) só funciona no navegador — abra o `07-dom/index.html`.

## Regras do jogo

1. Resolva os exercícios marcados com `✏️ EXERCÍCIO` direto no arquivo.
2. Rode o arquivo e confira se a saída bate com o esperado (está nos comentários).
3. Só abra o `GABARITO.md` depois de tentar de verdade.
4. Travou 15 minutos no mesmo exercício? Aí sim, olhe o gabarito, entenda, apague e refaça sozinho.
